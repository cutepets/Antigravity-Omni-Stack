# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [5.0.0] - 2026-04-02
### Khoác áo mới cho Antigravity Omni-Stack
- **Đại tu kiến trúc Multi-Agent**: Xác lập đội hình 16 Core Agents chia cụm chuyên biệt (Frontend, Backend, Design, Architect, DevOps, QA, Security, v.v.).
- **Mở rộng kho Kỹ năng (Skills)**: Đạt mốc 344 Skills chuyên sâu, tách rời khỏi cốt lõi Agent để giảm tải Context Window.
- **Quy trình chuẩn tắc (Workflow & Rules)**: 84 Slash Commands thông qua 37 Hiến pháp hệ thống giám sát.
- **Bảo mật tuyệt đối (AgentShield)**: Đạt chứng nhận Security Score Grade A (100/100) — gỡ bỏ triệt để điểm nguy hiểm về việc cấp phát đồng thời Full Access (Read/Bash/Write/Edit).
- **Hooks an toàn (Pre-Tool Use & Stop)**: Tích hợp `gsd-prompt-guard.js`, `gsd-workflow-guard.js` tự động ngăn lỗi.

## [4.x] - Quá khứ
