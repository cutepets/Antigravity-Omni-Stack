# 🚀 Antigravity Agent Framework

**Version:** 3.0.0 (Antigravity Omni-Stack Core - 100% Optimized)
**Core Engine:** Antigravity IDE & Model Context Protocol (MCP)

This is an internal **Agentic AI Coding Assistant** framework, fine-tuned to absolute perfection to handle the full lifecycle of software development, designing, coding, testing, and operations according to the highest industry standards. Rather than relying on a generic virtual assistant, this project utilizes a **Multi-Agent System** that operates on a "Divide & Conquer" philosophy.

## 🔄 3. Standard Workflows (Slash Commands)

Workflows are closed-loop, quality-assured processes that provide disciplined guidance for AI behavior.

Users trigger a Workflow by calling a **Slash Command** directly **inside the AI chatbox** (Do NOT run these in the Terminal):

*Core Workflows Example:*
- **`/plan`**: Map out the structural design from requirements before writing any code.
- **`/create`**: Ignite the code engine - rapidly scaffold modules / foundational features.
- **`/test`**: Auto-generate and run Unit/E2E Tests following strict TDD standards.
- **`/debug`**: Multi-threaded log scanning to trace the true Root Cause.
- **`/audit`**: Deep security, architectural, and performance audits prior to deployment.
- **`/release-version`**: Update Version, generate Changelog, and synchronize documents.

## 🛡️ 4. Core Rules (Constitution)

All Agent activities are strictly bound by rules constituted in the `.agent/rules/` directory (**Purged to only 37 core rules**). All AI decisions must pass through these constraints:
1. **GEMINI.md (Constitution):** Customizes work mindset, identifying Orchestrators and Scale-Awareness. Strictly enforces the **"Wait for Text Protocol"** to receive commands safely.
2. **SECURITY.MD:** Absolute zero-tolerance for Hardcoded Secrets, SQL Injections, and XSS.
3. **RUNTIME-WATCHDOG.MD:** Automated Stop & Clean mechanisms upon discovering infinite Hang / Loops.
4. **ERROR-LOGGING.MD:** Captures failures into an `ERRORS.md` file to act as the Agent's foundational learning document.
5. **MALWARE-PROTECTION.MD:** Blocks malicious links and strictly self-scans Subresource Integrity (SRI).
