## Current Interpretation

- Antigravity is the governance and routing layer.
- Specialist agents are operational playbooks.
- Skills are supporting deep methods loaded only when relevant.
- Workflows provide the structured path for heavy or risky tasks.
- GitNexus is the primary code intelligence layer.

## Routing Rule Of Thumb

- backend-heavy and code-heavy -> backend specialist or Codex-led backend flow
- UI-heavy and polish-heavy -> frontend specialist or Antigravity-led frontend flow
- ERP, architecture, or state-machine changes -> structured planning first
- review, verification, and security -> dedicated verification specialists
